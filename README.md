# 每日艺术 (Daily Art)

一个简洁、优雅的每日艺术鉴赏网页应用。它旨在提供一个沉浸式的、无干扰的艺术品浏览体验。

## ✨ 功能特性

- **全屏沉浸式浏览**: 每个艺术品占据整个屏幕，让你专注于作品本身。
- **动态内容加载**: 应用从芝加哥艺术博物馆 (Art Institute of Chicago) 的公共 API 动态获取艺术品数据，内容丰富多样。
- **随机探索模式**: 每次打开或刷新，都会呈现一个全新的、随机排序的艺术品列表，充满探索的惊喜。
- **智能翻译**: 自动将艺术品的标题和描述从英文翻译成中文，扫除语言障碍。
- **深入挖掘**: 看到喜欢的艺术家？点击右上角的“查看类似”按钮，即可深入探索该艺术家的更多作品。
- **流畅的交互**:
    - 支持类似抖音的上下滑动切换作品。
    - 优雅的加载动画和页面切换效果。
    - 响应式设计，在手机和电脑上均有出色的浏览体验。
- **优雅的设计**:
    - 采用明亮的浅色主题。
    - 精心挑选的艺术字体 (Playfair Display & Lora)。
    - 图片与文字信息分离的现代化布局，避免互相遮挡。

## 🚀 如何运行

1.  克隆或下载此项目到本地。
2.  在浏览器中直接打开 `index.html` 文件即可开始浏览。

## 🛠️ 技术栈

- **前端**: HTML, CSS, JavaScript
- **数据来源**: [Art Institute of Chicago API](https://api.artic.edu/docs/)
- **翻译服务**: [MyMemory Translation API](https://mymemory.translated.net/doc/spec.php)
- **字体**: [Google Fonts](https://fonts.google.com/) (Playfair Display, Lora)
